    Habitualmente la enseñanza de la trigonometría se justifica mediante situaciones que se pueden modelar con triángulos rectángulos y en las que se necesita conocer la medida de algunos de sus lados, pero que no son accesibles para medir directamente, como alturas de montañas, pirámides o en Astronomía. 

    \subsection*{Guía de Fórmulas}

    \textbf{1. Razones trigonométricas en triángulos rectángulos}

    \[
    \sin(\theta)=\frac{\text{cateto opuesto}}{\text{hipotenusa}}
    \]

    El seno de un ángulo en un triángulo rectángulo se define como el cociente entre el cateto opuesto y la hipotenusa. Permite calcular alturas cuando se conoce el ángulo y la longitud del lado inclinado.

    \[
    \cos(\theta)=\frac{\text{cateto adyacente}}{\text{hipotenusa}}
    \]

    El coseno de un ángulo es la razón entre el cateto adyacente y la hipotenusa. Se utiliza para relacionar distancias horizontales con la longitud de la hipotenusa.

    \[
    \tan(\theta)=\frac{\text{cateto opuesto}}{\text{cateto adyacente}}
    \]

    La tangente relaciona el cateto opuesto con el cateto adyacente. Es especialmente útil para calcular la altura de un objeto cuando se conoce la distancia horizontal y el ángulo de elevación.

    \vspace{0.3cm}

    \textbf{2. Cálculo de la altura mediante trigonometría}

    \[
    h=d\cdot\tan(\theta)
    \]

    Si \(d\) es la distancia horizontal al edificio y \(\theta\) el ángulo de elevación, la altura \(h\) se obtiene multiplicando la distancia por la tangente del ángulo. Este es el método más directo para calcular alturas inaccesibles.

    \[
    h=L\sin(\theta)
    \]

    Cuando se conoce la distancia inclinada \(L\) hasta la parte superior del edificio, la altura puede calcularse multiplicando esa distancia por el seno del ángulo. Este método utiliza la relación entre la hipotenusa y el cateto opuesto.

    \[
    d=L\cos(\theta)
    \]

    El coseno permite calcular la distancia horizontal a partir de la hipotenusa y el ángulo de elevación. Puede utilizarse para comprobar resultados obtenidos mediante otros procedimientos trigonométricos.

    \vspace{0.3cm}

    \textbf{3. Comprobación mediante el Teorema de Tales}

    \[
    \frac{h_1}{d_1}=\frac{h_2}{d_2}
    \]

    El Teorema de Tales establece que dos triángulos semejantes mantienen proporciones iguales entre sus lados correspondientes. Esto permite calcular la altura de un objeto comparándolo con otro de altura conocida.

    \vspace{0.3cm}

    \textbf{4. Conversión de grados a radianes}

    \[
    \theta_{\text{rad}}=\theta_{\text{deg}}\cdot\frac{\pi}{180}
    \]

    Esta fórmula permite transformar una medida angular expresada en grados a radianes. Es necesaria porque muchas funciones trigonométricas y cálculos matemáticos utilizan el sistema de radianes.

    \[
    \pi \ \text{rad}=180^\circ
    \]

    Esta equivalencia relaciona los dos sistemas de medida angular más utilizados. Surge de la definición del radián basada en la longitud del arco de una circunferencia.

    \[
    \theta=\frac{s}{r}
    \]

    Un radián se define como el ángulo central que abarca un arco de longitud igual al radio de la circunferencia. En la fórmula, \(s\) representa la longitud del arco y \(r\) el radio.

    \vspace{0.3cm}

    \textbf{5. Cálculo del error porcentual}

    \[
    \text{Error \%}=\frac{|V_{\text{experimental}}-V_{\text{real}}|}{V_{\text{real}}}\cdot100
    \]

    El error porcentual mide la diferencia relativa entre un valor obtenido experimentalmente y el valor real o de referencia. Permite evaluar la precisión de las mediciones realizadas.

    \[
    h=d\cdot\tan(\theta\pm1^\circ)
    \]

    Esta expresión permite analizar cómo cambia la altura calculada si el ángulo medido tiene un posible error de \(\pm1^\circ\). Sirve para estudiar la sensibilidad del resultado ante pequeñas variaciones en la medición.

    \subsection*{Descripción del trabajo}
        En grupos de 3/4 alumnos deberemos calcular las dimensiones del instituto y presentar una memoria estructurada.



        \textbf{Fase 1: Preparación}
            \begin{itemize}
                \item Instalar app clinómetro: permite medir ángulos.
                \item Recordar las razones trigonométricas en triángulos rectángulos.
                \item Repasar la conversión de grados a radianes.
            \end{itemize}

        \textbf{Fase 2: Medidas}
            \begin{itemize}
                \item Elige un edificio alto del  entorno
                \item Medir la distancia horizontal del edificio al ángulo.
                \item Medir el ángulo de elevación
            \end{itemize}

        \textbf{Fase 3: Cálculo médiente distintos métodos}
            \begin{itemize}
                \item Aplicar la fórmula de la tangente para calcular la altura.
                \item Resolver el mismo problema utilizando seno y coseno.
                \item Comprobar la altura utilizando el Teorema de Tales.
                \item Comparar los resultados obtenidos.
            \end{itemize}

        \textbf{Fase 4: Trabajo en radianes}

            \begin{itemize}
                \item Convertir el ángulo medido a radianes.
                \item Expresar el valor en función de $\pi$.
                \item Repetir los cálculos utilizando la calculadora en modo radianes.
                \item Explicar qué representa un radián y por qué $\pi$ rad $= 180^\circ$.
            \end{itemize}

        \textbf{Fase 5: Justificación teórica}
            \begin{itemize}
                \item Dibujar el triángulo rectángulo asociado.
                \item Relacionar seno y coseno con la circunferencia goniométrica.
                \item Explicar por qué la tangente crece rápidamente cuando el ángulo se aproxima a $90^\circ$.
            \end{itemize}

        \textbf{Fase 6: Análisis del error}
            \begin{itemize}
                \item Comparar los resultados obtenidos por los distintos métodos.
                \item Calcular el error porcentual entre métodos.
                \item Analizar cómo afecta un posible error de $\pm 1^\circ$ en la medición del ángulo.
            \end{itemize}

        




    \subsection*{¿Qué hay que entregar?}

    \begin{enumerate}
        \item Datos del grupo (integrantes, fecha, objeto medido)
        \item Planteamiento del problema: Explicamos qué queremos calcular y por qué
        \item Datos que se han recogido
        \item Esquema gráfico (catetos, hipotenusas, ángulos)
        \item Desarrollo matemático
        \item Resultado final: ¿Altura del objeto:  ¿? Metros
        \item Reflexión final: ¿Ha sido preciso el método? ¿Qué posibles errores pueden aparecer? ¿Cómo mejoraríamos la medición?
    \end{enumerate}
